## Remote Setup

Use these remotes to keep the fork and its upstream in sync:

```bash
git remote add origin https://github.com/dimaslanjaka/opencode-file-memory.git || git remote set-url origin https://github.com/dimaslanjaka/opencode-file-memory.git
git remote add upstream https://github.com/joshuadavidthomas/opencode-agent-memory || git remote set-url upstream https://github.com/joshuadavidthomas/opencode-agent-memory
```
